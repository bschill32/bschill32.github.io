Slackin

Technology Used

React.js | Node.js | Pusher | Electron | Express | PostreSQL | Material-UI | SASS | HTML5 | CSS3

Slackin is a real-time chat application along the lines of Slack or Discord. It features a flat channel structure, where any user can create a chat channel which other users can see, join, and subscribe to. Users can also send direct messages and form new relationships with each other.
